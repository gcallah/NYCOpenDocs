The startup folder is used to launch applications upon boot.
Celery is an asynchronous task queue and a Flask application that uses
Celery initializes the Celery client to run the application.
The three components of Celery installation are the Celery client, Celery
workers and the message broker.
In order to start the celery workers both a Celery worker and a Beat instance
need to run in parallel.
