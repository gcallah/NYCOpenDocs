This JSON file contains an object with the property "agencies", which is mapped to a list of agencies. Each agency is defined as an object with its associated properties.
<br>
Each agency has these properties:
<ul>
<li>Category</li>
<li>Name</li>
<li>Category</li>
<li>Default Email</li>
<li>Appeals Email</li>
<li>Features - Letter</li>
<li>
Custom Request Form
    <ul>
    <li>Each agency can have the custom request form enabled or disabled. Other attributes associated with the request form pertain to whether the form is expanded by default, is categorized, is description hidden by default, etc.</li>
    </ul>
</li>
<li>Agency Requests</li>
<li>Specific Request Instructions</li>
<li>Acronym</li>
</ul>
